## 📖 About

This repository contains my accepted solutions to problems solved on **Codeforces**.

The goal is not only to store accepted solutions but also to create a personal knowledge base for revision and interview preparation.

Each problem includes:

- ✅ Original Problem Statement
- ✅ Optimized C++ Solution
- ✅ Detailed Explanation
- ✅ Multiple Approaches (when applicable)
- ✅ Dry Run
- ✅ Time Complexity
- ✅ Space Complexity
- ✅ Key Observations
- ✅ Edge Cases
